Chess

Usage:

/chess start - start a new game/chess join - join a started game /move <move e.g. e2e4>/chess resign - resign from the current game /chess board - show the current board/chess history - show the move history /chess undo request - request to undo the last move /chess undo <accept/reject> - accepts or rejects an undo request

Alternatives


/move - /chess move

Description

Simulates a simple chess game.
